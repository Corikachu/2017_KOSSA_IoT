기간 : 17.10.16 ~ 17.10.20

주제 : 벌레로 인해 고통받는 전우들을 위한 효율적인 벌레 예방

개요 : 군인이라면 매일 같이 하게 되는 청소, 그런데 저녁 청소 시간에 환기를 위해 창문을 열면 방충망이 있어도 벌레가 많이 들어온다.
         그래서 창문 앞에 벌레퇴치기를 설치하고 이를 효율적으로 작동시키기 위해 아두이노를 활용해보겠다!

기본 배경 : 창문에 초음파 센서를 달아서 창문이 어느정도 이상 열리면 서보모터를 활용하여 벌레 퇴치기가 켜지도록 하고 창문이 어느정도 이상 닫히면 벌레 퇴치기가 꺼지도록 한다.
                그리고 해가 진후에만 켜질 수 있도록 조도 센서를 활용하여 설정을 한다.

개발 환경에 알맞게 수정 > 초음파 센서 (기본키트 구성품에 없음) -> 푸시 버튼 / 푸시 버튼이 창문에 의해 눌려있는 동안 켜지고 푸시 버튼에서 떨어지면 꺼지는 방식 (푸시 버튼의 문제점 : 누르는 상태가 유지되지 않음) -> 한번 누를때마다 켜지고 꺼지는 방식 / 조도 센서의 활용이 무의미해짐 + 조도센서의 작동불량 -> 조도 센서 활용 제외

개선된 배경 : 창문 바로앞에 설치되어있는 벌레 퇴치기를 효율적으로 작동시키기 위해 창문 옆에 푸시 버튼을 달고 창문을 열고 닫을 때 이 버튼을 이용해 벌레 퇴치기를 켜고 끌 수 있도록 한다.

설계도 : (block diagram) in PPT

개선점 :
1) 초음파 센서를 활용한다면 더 효율적이게 사용할 수 있을 것이라고 생각된다. -> 버튼 누르지 않아도 자동으로 온/오프 가능


발전 가능성 & 또 다른 아이디어 :
1) 조도 스위치를 활용하여 야간 상황 조건을 받아 스위치가 켜지거나 꺼지도록 설정할 수 있을 것 같다. -> ex) 청소 시간 외에도 야간에 지속적으로 벌레 퇴치기 가동,  해가 진 후 가로등 점등...

2) 초음파 센서를 활용하여 근처에 접근시 장비가 작동하도록 하는 제품을 만들 수 있을 것 같다. -> 요즘 나오는 전자 제품 중에는 초음파 센서를 내장하고 자동으로 온/오프 해주는 방식을 사용하는 경우도 있지만 이미 시중에 나와있는 많은 전자 완제품의 경우에는 물리적으로 온/오프 버튼을 눌러주는 방식이 필요하다. 따라서 스위치마다 필요한 모터를 표준화하면 간단하게 접근시 켜지고 꺼지는 방식을 사용할 수 있을 것 같다. 

3) 전기방충망 - 원래는 전기 방충망을 만들고 싶다는 생각으로 아이디어에 접근했다. 그런데 아두이노 우노보드로는 전압을 방출할 수가 없다는 사실을 알게 되고 계획을 수정했다. 나중에 기회가 된다면 야간에 문이 열릴때만 전기방충망이 작동하도록 만들고 싶다. (전기 방충망은 2중 철망으로 또 따로 제작해야함)
